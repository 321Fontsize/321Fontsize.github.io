# 管理

参考知网文献导出（导出与分析 :arrow_right: 导出文献 :arrow_right: 自定义 :arrow_right: 下载xls），在某个时间段内，我们看了若干篇论文，可以将其按如下方式汇总：

作者 + 题名 + 关键词 + 摘要

# 阅读

每一篇论文做好读书笔记。

PDF命名清晰：作者\_年份\_关键词（如Raissi\_2019\_PINNs.pdf）。

**阅读顺序**：

摘要/引言 → 结论 → 问题综述与背景（略读） → 方法概述 → 重点技术细节

**读书笔记模板示例**：

```
标题：Physics-Informed Neural Networks（2019）
问题：求解非线性微分方程正逆向问题
方法：深度学习框架、神经网络
创新：将偏微分方程嵌入神经网络的损失函数之中
启发：极大地摆脱传统CFD对网格的依赖
状态：已读完，并复现其中部分代码
```
